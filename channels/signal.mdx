## Overview

The Signal channel integrates IronClaw with Signal messenger via signal-cli.

## Prerequisites

- signal-cli daemon running
- Signal account (phone number)
- IronClaw configured with Signal credentials

## Configuration

```bash
export SIGNAL_HTTP_URL=http://127.0.0.1:8080
export SIGNAL_ACCOUNT=+1234567890
export SIGNAL_ALLOW_FROM=+0987654321,+11234567890
```
